集装箱箱型尺寸代码
本标准根据ISO／TC104集装箱标准化技术委员会起草和发布的集装箱国际标准而编制。本标准引用ISO6346：1995《集装箱--代码、识别和标记》的版本是在1997年10月获得各国投票通过的最新版本。使用者在引用该国际标准时需注意ISO正式颁布和实施的日期。本标准是国际集装箱运输EDI标准体系交通行业系列标准之一。ISO6346规定的集装箱类型和尺寸代码作为国际集装箱设备的必备标识符；应与箱主代码和箱号等必备标识符组配标识，用于
国际集装箱运输EDI信思标识、处理与交换。
本标准的附录A和附录B是提示的附录。
本标准由中华人民共和国交通部提出并归口。
本标准起草单位 ：交通部标准计量研究所 、交通部水运科学研究所
本标准主要起草人 ：孙黎莹 张静轩 华晓江
中华人民共和国交通行业标准

1.范围
本标准等效采用ISO 6346《集装箱--代 码 、识别和标记》规定的系列1通用、专用集装箱类型和尺寸代码。
本标准适用于从事国际集装箱运输的系列1标准集装箱的设备标识和EDI报文等信息的处理与交换。
2.引用标准
下列标准包含的条文，通过在本标准中引用而构成为本标准的条文。在标准出版时，所示版本均为有效。所有标准都会被修订，使用本标准的各方应探讨使用下列标准最新版本的可能性。
ISO 668: 1995系列1集装箱--类型、外部尺寸和额定值。
ISO 830: 1995 集装--术语
ISO 1161: 1995 系列1集装箱
ISO 1496: 1990 系列1集装箱--技术条件和试验方法
ISO 6346: 1995 集装箱--代码、识别和标记
3.分类原则
3.1 集装箱类型按运输方式、货物类别，箱体特征等原则分类。
3.2 集装箱尺寸按箱体的外部长度，宽度和高度公称尺寸和最小内部尺寸分组，对应于系列1集装箱的各类箱型代码。
4.代码结构
4.1 集装箱类型代码引用ISO 6346的第4.2.2条的规定。箱型代码结构由两位字符表示：
X X
- -
| |
| ---箱型编号
----------箱型标识符
4.2 集装箱尺寸代码引用ISO 6346 的附录D的规定。尺寸代码结构由两位字符表示：
X X
- -
| |
| ---箱高和箱宽代码
----------箱长代码
注: 第1位由一个字符表示箱长；
第2位由一个字符表示箱宽和箱高。

6.使用说明
6.1 按照ISO 6364中第6章“标记的标志方法”，箱型和尺寸代码应作为一个整体在集装箱上标识。其组配代码结构为：
XX XX
-- --
| |
| ---箱型代码
----------尺寸代码
例如: 22G1指箱长为20英尺(6068mm)，箱宽为8英尺(2438mm)和箱高为8英尺6英寸(2591mm)，上方有透气罩的通用集装箱。
6.2 本标准附录A（ 提 示 的 附 录 ）中给出部分国内集装箱类型与尺寸的查询参考表。
由于ISO 6346己作修订，1995年版修订后的箱型比1984年版的箱型代码标识改变较大，使用中应注意记录实际箱体上标识的代码与编号；新生产的集装箱则应标注ISO 6346新版本的箱型代码。
6.3 在记录箱体信息时，应将类型。尺寸代码与箱主代码，箱号等信息组配标识。本标准在附录B(提示的附录）中给出ISO 6346第6章规定的集装箱识别标记及其代码的布置图样。
